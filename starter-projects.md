# Starter Projects (OSINT and Cybersecurity)

These projects are designed to help beginners apply OSINT and cybersecurity concepts in practical, real-world scenarios. Each project focuses on building core investigation and analysis skills.

---

## Project 1: OSINT Investigation

### Description
Conduct an investigation using only publicly available information on a person, organization, or topic.

### Steps
1. Define a clear objective (e.g., identify public information about a company)
2. Use search engines and OSINT tools to gather data
3. Identify key details such as:
   - names
   - locations
   - domains
   - associated accounts
4. Verify findings using multiple sources
5. Document results in a structured format

### Skills Developed
- Data collection  
- Verification and cross-referencing  
- Analytical thinking  

---

## Project 2: Passive Reconnaissance on a Domain

### Description
Perform passive reconnaissance on a domain without directly interacting with the target.

### Steps
1. Choose a public domain (e.g., a company website)
2. Use tools such as:
   - Google Dorking
   - Shodan
   - Amass
3. Gather information such as:
   - subdomains  
   - exposed services  
   - publicly available documents  
4. Organize findings into categories
5. Identify patterns or potential exposure points

### Skills Developed
- Reconnaissance techniques  
- Understanding infrastructure  
- Data correlation  

---

## Project 3: Phishing Detection (Concept or Prototype)

### Description
Analyze phishing techniques and build a basic system or framework to identify phishing emails.

### Steps
1. Collect examples of phishing and legitimate emails (public datasets if possible)
2. Identify common features:
   - suspicious links  
   - language patterns  
   - sender inconsistencies  
3. Create a simple detection approach:
   - rule-based checklist OR  
   - basic machine learning model  
4. Test the approach against sample data
5. Document results and limitations

### Skills Developed
- Threat analysis  
- Pattern recognition  
- Introduction to machine learning and NLP  

---

## Project 4: Identity Pivoting (Using Melissa and OSINT Tools)

### Description
Start with a single data point (email, phone number, or name) and expand it into a broader profile using OSINT tools.

### Steps
1. Begin with one identifier (e.g., email address)
2. Use Melissa to:
   - validate the data  
   - extract structured information  
3. Pivot into:
   - search engines  
   - social media tools  
   - domain lookups  
4. Correlate findings across sources
5. Build a basic profile of the entity

### Skills Developed
- Data enrichment  
- Pivoting techniques  
- Correlation and analysis  

---

## Why These Projects Matter

These projects simulate real-world OSINT and cybersecurity workflows. They help build practical skills that are directly applicable to roles in threat intelligence, security operations, and investigative analysis.
